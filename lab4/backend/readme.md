# Configure

 1. You need to create a .env at `/` with your private data.
Sample:
```
   DB_HOST="example.com"
   DB_USER="user"
   DB_NAME="nameofbase"
   DB_PASSWORD="qwerty"
   WEATHER_API_KEY="6g88dsa6dbs7"
```
2. You need to have `favcity` table in DB with primary key `NAME` -- varchar
3. run `node index.js` to start the server
